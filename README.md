# Проект: Место

### Это одностраничный сайт. На этом сайте собранны карточки различных мест в России. Пользователь может редактировать свой профиль. В дальнейшем будет больше функционала.
### Использованны следующие технологии:
* grid и flexbox
* JavaScript
* адаптивная верстка
